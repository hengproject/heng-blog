# Agent project notes

Use the files in this directory as project-specific operating context:

- [`COMPONENTS.md`](./COMPONENTS.md): custom component selection and MDX usage.
- [`TODO.md`](./TODO.md): temporary assets, placeholder content, and unfinished configuration.
- [`release/SKILL.md`](./release/SKILL.md): release entry point. Run its stages in order.

Component implementation details and the complete prop tables remain next to the source files in
`src/components/custom/`. Treat those component-local documents as the source of truth and keep
`COMPONENTS.md` synchronized when a component's public usage changes.
